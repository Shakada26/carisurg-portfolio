## 1. Project Overview

This repository contains my structured learning portfolio completed as part of the CariSurg MedTech Pathways Programme.

The project focuses on clinical AI, emergency department triage analytics, healthcare data science, and reproducible research using Git and GitHub.

---

## 2. Clinical Context

Emergency departments require rapid triage decisions under high-pressure conditions. Challenges include:

- Missing clinical data  
- Inconsistent recording of vital signs  
- High patient volumes  
- Variability in clinician judgement  

This project explores how AI decision-support tools can improve consistency while maintaining clinician oversight.

---

## 3. Dataset Description

The dataset is a reduced and de-identified emergency department triage dataset used for educational purposes.

### Data includes:
- Demographics  
- Vital signs (pulse, temperature, etc.)  
- Triage-related clinical indicators  

### Ethics:
- No patient-identifiable data  
- Educational use only  
- Not publicly redistributed
